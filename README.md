
# 📰 Fake News Detector

An AI-powered fake news detection system using TF-IDF and Logistic Regression.

## 📊 Results
- Accuracy: 99%
- Precision: 99% | Recall: 99% | F1-Score: 99%
- Trained on 44,898 real and fake news articles

## 🛠️ Technologies Used
- Python
- Scikit-learn (TF-IDF + Logistic Regression)
- Gradio (Web Interface)
- Google Colab (CPU)
